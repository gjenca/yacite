USAGE: yacite append [-h] fieldname string [string ...]

DESCRIPTION:

reads YAML stream, appends all strings in the list to the value of a field, outputs YAML stream

POSITIONAL ARGUMENTS:

fieldname

Field name. Value must be a 'list of strings', if it does exist. If it does not, it is created.

string

these strings are appended to the value

EXAMPLES: 

yacite append tags scopus foreign
yacite append grants my_grant_number
yacite append keywords "vector spaces"
