USAGE: yacite unappend [-h] fieldname string [string ...]

DESCRIPTION:

reads YAML stream, deletes each string in the list from the list of strings in the value of a field, outputs YAML stream

POSITIONAL ARGUMENTS:

fieldname

Value of the field must be 'list of strings'.

string

these strings are removed from the value, if present

EXAMPLE:

yacite unappend tags scopus sci
yacite unappend keywords mathematics
