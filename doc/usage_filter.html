USAGE: yacite filter [-h] [--myown] [--notmyown] [-f] [-m MODULE] [-k] expr

DESCRIPTION:

reads YAML stream, evaluates a python expression in the context of the each record, outputs YAML stream with records for which expression returns True

POSITIONAL ARGUMENTS:

expr

python expression

OPTIONAL ARGUMENTS:

--myown

filter applies only if myown == True, otherwise the record passes through

--notmyown

filter applies only if myown == False or undefined, otherwise the record passes through

-f, --failed

output only the failed records,supress error message

-m, --module MODULE

python module to import

-k, --keep-going

do not stop when the eval(expr) throws an exception

EXAMPLES: 

yacite filter '"scopus" in tags'
yacite filter 'year>2010'
yacite filter 'year<2000 and "Hilbert space" in title'
yacite filter '"grants" in dir()'
yacite filter '"NSF" in grants'
